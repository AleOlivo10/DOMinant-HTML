Facts:

  • The DOM is a tree
  • HTML tags are also Element Nodes or just Elements
  • You access the DOM components via its elements
  • Elements have parents, children, siblings, ancestors, and descendants.
  • Selector a: getElementById (only one #ID)
  • Selector b: getElementsbyClassName (several things inside #classes)
